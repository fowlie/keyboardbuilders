# Keyboard Builders

A web application for keyboard enthusiasts to manage and share their keyboard builds.

## Features

- User authentication with Auth0
- Create and manage keyboard builds
- Modern UI with Semantic UI React
- Full-stack application with React frontend and Java backend

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

## Technologies Used

- Frontend: React, Vite, Semantic UI React
- Backend: Java Spring Boot
- Authentication: Auth0
- Database: (Your database choice)
